# Dino Dash

A 3D platformer game built in Unity where a dinosaur collects apples and coins across 4 levels.

## About the Game
Dino Dash is a 3D platformer mobile game. The player controls a dinosaur and must collect apples to restore health and coins to increase score. Each level has obstacles to avoid and a goal to complete.

## Gameplay
- Collect **apples** to restore health
- Collect **coins** to increase score
- Avoid **obstacles** to stay alive
- Complete all **4 levels**

## Features
- 4 playable levels
- Health system
- Score system
- Obstacle avoidance
- Level progression
- Mobile-ready controls

## Built With
- **Engine:** Unity
- **Language:** C#
- **Platform:** Mobile (Android)
